Linux下软件的安装

很多软件需要自己下载源代码，自己编译，因为很多发行版都有自己的安装包格式，因此很多发行版中都
提供了类似于应用市场、Nuget的东西。Ubuntu推荐使用apt进行下载、安装。
sudo apt/apt-get install 程序名
sudo apt/apt-get remove 程序名
（sudo是使用管理员权限执行命令）

 
查看网络配置ifconfig

SSH服务：运维人员远程连接到机房上的服务器，使用命令行操作服务器。
相比于Telnet协议的明文传输数据，SSH通讯过程数据是加密的。

Linux安装使用OpenSSH
Ubuntu默认值安装了客户端openssh-client
安装服务器：sudo apt install openssh-server
查看服务器是否开启：netstat -tlp(显示已启动的服务)
启动 sudo/etc/init.d/ssh start
重新启动 sudo/etc/init.d/ssh restart
ssh服务默认端口22 可通过修改配置文件修改端口etc/ssh/sshd_config
注意linux虚拟机的网卡设置：桥接网卡==>主机网卡

windows下如果通过ssh连接到linux：
下载运行putty，免安装。
如果有乱码，可以在translation中更改为utf-8
在putty中全命令行操作远程服务器资源

安装Mysql服务器
sudo apt install mysql-server   提示输入用户名密码
安装Mysql客户端
sudo apt install mysql-client
使用客户端登录服务器
mysql -uroot -p123
show databases;
select now();
quit;退出mysql

基于安全考虑，mysql默认只能本机连接，如果想远程访问，就需要配置。
最好不要配置成任意电脑都可以连接，风险大，要配置成只允许某些ip访问。


配置MySQL远程访问（需要配置为桥接网络）


sudo vi /etc/mysql/mysql.conf.d/mysqld.cnf
由于mysql版本不同可能配置文件存放位置有所区别
使用cd、dir一级级的查看

编辑修改配置文件：
注释行bind address 127.0.0.1
保存退出wq
执行重启操作
sudo /etc/init.d/mysql/restart
查看主机ip
进入linux的mysql客户端中执行语句：
grant all privileges on *.* to root@"主机IP" identified by "数据库密码" with grant option;
再执行
flush privileges;


安装vsftp
sudo apt install vsftpd

推荐使用的ftp客户端filezilla ftp client

登录使用虚拟机ip、用户名、密码、默认端口 即可连接

默认开始可以用linux用户登录，访问/home/用户名的文件夹下，但是只能下载不能上传。
需要修改配置文件
sudo vi /etc/vsftpd.conf
将"write_enable=YES"前面的#注释取消
然后重启服务
sudo /etc/init.d/vsftpd restart 重启

如果上传或下载很多的文件，建议先打包再解压缩
upzip xxx.zip
upzip abc\?.zip
zip -r xxx.zip xxx.txt dir1
upzip -v large.zip 不解压查看
upzip -t large.zips 想看看下载完全了吗

***通过命令行创建多项目解决方案
步骤如下：
1）先创建解决方案目录pianobbs,然后在其中执行命令 dotnet new sln
2) dotnet new mvc -o pianobbs.web 说明:创建web项目
3）dotnet new classlib -o pianobbs.common 说明:创建common类库项目
4) dotnet sln pianobbsTest1.sln add pianobbsTest1.web/pianobbsTest1.web.csproj
说明：将web项目添加到解决方案中
5) dotnet sln pianobbsTest1.sln add pianobbsTest1.common/pianobbsTest1.common.csproj
说明：将common项目添加到解决方案中
6) dotnet add pianobbsTest1.web/pianobbsTest1.web.csproj reference pianobbsTest1.common/pianobbsTest1.common.csproj
说明：添加web项目对common项目的引用
（可通过cat pianobbsTest1.sln和cat pianobbsTest1.web/pianobbsTest1.web.csproj查看项目添加和引用结果）
7) dotnet restore 通过nuget还原加载整个解决方案的各个项目中要使用的包
8) cd pianobbsTest1.web 进入到主项目中
9) dotnet run 运行主项目
